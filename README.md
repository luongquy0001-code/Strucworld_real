Mục đích:
Dự án này chứa các script demo/đồ chơi và một vài mô-đun thử nghiệm (Python + một file C++ demo). Mục tiêu: học tập và thử nghiệm.

Quick start:
1. Clone:
   git clone https://github.com/luongquy0001-code/Strucworld_real.git
2. Mở file .py bằng editor và đọc trước khi chạy.
3. Nếu muốn chạy, tạo virtualenv và chạy các script test an toàn.
